# Skill: Angular UI Kit docs updater

## Quando usarla
Usa questa skill quando viene aggiunto o modificato un componente della UI kit e serve aggiornare la documentazione in `docs/components`.

## Checklist
1. Individua la categoria corretta (`foundation`, `interactive`, `advanced-inputs`).
2. Aggiorna il markdown con:
   - breve descrizione del componente;
   - snippet Angular (`imports` + template);
   - note a11y e keyboard.
3. Mantieni naming e terminologia coerenti con `UiXxxComponent`.
4. Aggiungi eventuali TODO tecnici in fondo al file docs.
